Notification Center

A supervisor tool for real-time agent communication in Dynamics 365

What is this app?

The Notification Center allows supervisors to create, schedule, and send real-time notifications directly to agents working in Dynamics 365. Agents receive pop-up alerts with optional acknowledgment tracking, so you always know who has seen your messages.

How to use it

1. Create categories — Go to the Categories tab to organize your notifications by type (e.g., Urgent, Policy Update). Assign an emoji and color to each.

2. Compose a notification — In the Notifications tab, click + New. Write a title and body, pick a category, choose priority, and optionally target specific queues.

3. Send or schedule — Send immediately or set a future date/time for automatic delivery.

4. Track acknowledgments — Enable Require Acknowledgment to see which agents have confirmed reading the notification.

5. Review reports — Use the Reports tab for analytics on delivery, acknowledgment rates, and trends over time.
